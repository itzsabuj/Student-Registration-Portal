## 🚀 How to Start the Project
To run the student registration portal locally:

1. Navigate to the `html` directory.
2. Open the `dom_task.html` file in your browser.
3. Enter a student detail
4. Click on Register for student registration.
5. Reset or Delete any data of student, if required
 by clicking the respective buttons.

Github link - https://github.com/itzsabuj/Student-Registration-Portal.git
